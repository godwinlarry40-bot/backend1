# TradePro Backend

A comprehensive cryptocurrency trading and investment platform backend built with Node.js, Express, and MongoDB.

## Features

- **User Authentication & Authorization**: Secure JWT-based authentication with role-based access control
- **Portfolio Management**: Track and manage cryptocurrency portfolios
- **Wallet System**: Multi-currency wallet with deposit/withdrawal functionality
- **Real-time Market Data**: Integration with CoinGecko API for live cryptocurrency prices
- **Investment Platform**: Create and manage investment plans with automatic profit calculation
- **Transaction History**: Complete record of all financial transactions
- **Security Features**: Rate limiting, input validation, encryption, and secure password handling
- **Email Notifications**: Automated emails for important events

## Tech Stack

- **Runtime**: Node.js (ES Modules)
- **Framework**: Express.js
- **Database**: MongoDB with Mongoose ODM
- **Authentication**: JWT (JSON Web Tokens)
- **Caching**: Redis (optional)
- **API Integration**: CoinGecko, Blockchain APIs
- **Email**: Nodemailer
- **Logging**: Winston
- **Validation**: Express Validator
